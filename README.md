# N8N — Aula 7 TTI

Repositório para rodar o N8N via GitHub Codespaces.

## Como usar

1. Clique em **Code → Codespaces → Create codespace on main**
2. Aguarde o ambiente iniciar (~2 min na primeira vez)
3. O N8N abre automaticamente no browser
4. Crie sua conta local (qualquer e-mail e senha — os dados ficam só no Codespace)

> Os dados persistem enquanto o Codespace existir. Para exportar seu fluxo: menu **(⋯) → Download** → salva como `workflow.json`.
